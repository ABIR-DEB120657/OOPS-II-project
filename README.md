## рџ“– About The Project

**Nexus One** is a full-stack, role-based **Intelligent University Management System** developed exclusively for **Khwaja Yunus Ali University (KYAU)**. It digitizes and automates the core academic operations of the university вЂ” eliminating paper-based processes for attendance, finance, leave applications, assignments, and more.

The platform supports three dedicated portals: **Student**, **Teacher**, and **Admin** вЂ” each with a tailored set of features built to simplify everyday university life.

## вљ™пёЏ Getting Started

**1. Clone the repository**
```bash
git clone https://github.com/ABIR-DEB120657/OOPS-II-project.git
cd OOPS-II-project
```

**2. Build with Maven**
```bash
mvn clean install
```

**3. Run the application**
```bash
mvn spring-boot:run
```

**4. Open in browser**
```
http://localhost:8080
```
