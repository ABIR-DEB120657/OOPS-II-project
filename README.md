## 🚀 Core Features

<div align="center">
<a href="https://git.io/typing-svg">
  <img src="https://readme-typing-svg.demolab.com?font=Fira+Code&weight=600&size=16&pause=1200&color=4fc3f7&background=00000000&center=true&vCenter=true&width=600&lines=Explore+Student+Portal+Features...;Explore+Teacher+Portal+Features...;Explore+Shared+Utilities..." alt="Features Typing SVG" />

<br/>

Nexus One is divided into dedicated portals, each tailored with specific functionalities:

<details open>
<summary><h3>👨‍🎓 Student Portal &nbsp;<img src="https://img.shields.io/badge/Portal-Student-e94560?style=flat-square"/></h3></summary>

<br/>

| Feature | Description |
|:---:|:---|
| 🧠 **Smart Dashboard** | Track live CGPA goals, attendance rate, and urgent notices |
| 📊 **Academic Records** | Semester-wise transcript view and performance forecasting |
| 💳 **Finance & Fees** | Generate automated, sequential 3-copy payment slips (Trust Bank) |
| 🏖️ **Leave Applications** | Request leaves and track approval status directly |
| 📝 **Assignments & Materials** | Submit assignment files, view course outlines, and track deadlines |

</details>

<details open>
<summary><h3>👨‍🏫 Teacher Portal &nbsp;<img src="https://img.shields.io/badge/Portal-Teacher-4fc3f7?style=flat-square"/></h3></summary>

<br/>

| Feature | Description |
|:---:|:---|
| 📋 **Intelligent Attendance** | Real-time session attendance with low-attendance alerts |
| ✅ **Assignment Evaluator** | Create, manage, and grade student submissions easily |
| 🤝 **Faculty Sync** | Manage office hours and accept/decline student meeting requests |
| 🔐 **Leave Management** | Approve or reject student leave applications in one click |
| 📚 **Course Outlines** | Upload modules, PDFs, and resources for specific batches |

</details>

<details open>
<summary><h3>⚙️ Shared Utilities &nbsp;<img src="https://img.shields.io/badge/Shared-Utilities-6DB33F?style=flat-square"/></h3></summary>

<br/>

| Feature | Description |
|:---:|:---|
| 🩸 **Blood Finder Directory** | A built-in campus directory to request or donate blood during emergencies |
| 💼 **Career & Tuition Hub** | Official job circulars and student tuition vacancy postings |

</details>

## ⚙️ Getting Started / Installation

<div align="center">
  <img src="https://readme-typing-svg.demolab.com?font=Fira+Code&weight=600&size=15&pause=1000&color=6DB33F&background=00000000&center=true&vCenter=true&width=500&lines=Clone+→+Build+→+Run+→+Open+Browser+🚀" alt="Install Typing" />
</div>

<br/>

**1️⃣ Clone the repository**
```bash
git clone https://github.com/ABIR-DEB120657/OOPS-II-project.git
cd OOPS-II-project
```

**2️⃣ Build with Maven**
```bash
mvn clean install
```

**3️⃣ Run the application**
```bash
mvn spring-boot:run
```

**4️⃣ Open in browser**
```
http://localhost:8080
```
