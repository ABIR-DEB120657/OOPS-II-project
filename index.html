Key Features (Teacher Portal)

📊 Smart Dashboard
  • View classes scheduled for today
  • Quick actions (Attendance, Assignment, Leaves)
  • Monitor recent activity & pending reviews
  • View overall student attendance rate
✅ Session Attendance
  • Select Batch, Course, and Date
  • Auto-loads enrolled student list
  • Mark Present/Absent with real-time sync
  • Generates low-attendance warnings
📝 Assignment Evaluator
  • Create new assignments with deadlines
  • Track submission rates (e.g. 2/40 Submitted)
  • Identify overdue student submissions
  • Review code/files and assign grades
🤝 Faculty Sync
  • Manage designated Office Hours
  • Review consultation requests from students
  • Accept/Decline with Meeting/Chat links
  • View request urgency and topic context
